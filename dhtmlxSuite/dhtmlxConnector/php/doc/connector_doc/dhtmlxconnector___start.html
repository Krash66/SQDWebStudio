PHP Connector

DHTMLX connector is a helper server-side library that enables access to external datasources and backend systems.
It supports all DHTMLX components operated on data.
The library itself provides wide server-side functionality, such as data loading (static or dynamic), saving, filtration, validation etc. Cooperation with DHTMLX Dataprocessor allows you to manage insert/delete/update operations in addition.
